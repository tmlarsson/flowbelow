Import sensors via CSV file

The CSV file should be delimited with ";" and have the following form:

fid ; id ; Sensor ID ; Sensor typ ; Placement ; Masl. ; X ; Y ; Coordinate system

Where Sensor ID, X, Y, Masl. and Coordinate system are obligatory.

Note that the first row is skipped due to column names.
